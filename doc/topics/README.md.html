Get network information with Connman and DBus

This module provides a simple API to query Connman using DBus.

Requirements

In addition to the requirements specified in the rockspec file, you need Connman and DBus. Note that you may need permissions to access the Connman DBus interface.

Installation

This module can be installed with Luarocks by running

luarocks install connman_dbus

Use the --user option in luarocks if you don't want or can't install it system-wide.

Example

Below is a small example of how to use the module:

Manager = require("connman_dbus"):init()
print(Manager.OfflineMode) -- e.g. true
print(Manager.techs["/net/connman/technology/p2p"].Powered) -- e.g. false
service = Manager.current_service
print(service.Type) -- e.g. "wifi"
print(service.State) -- e.g. "online"
